Test the LHMP functionality

TestSample.Run()

Simple function for inputting camera settings and acquiring images from the LHMP.

Returns:

dictionary of raw image data in digital number along with the image metadata of exposure time (us), gain, acquisition time (UTC), sensor temperature (degC)

Return type:

numpy dictionary

Basic function for capturing samples with the LHMP

CaptureSample.Run(camera_settings)

Basic function for capturing samples with the LHMP.

Parameters:

camera_settings (numpy dictionary) – dictionary containing the gain and exposure time camera settings as well as the acquisition duration

Returns:

dictionary of raw image data in digital number along with the image metadata of exposure time (us), gain, acquisition time (UTC), sensor temperature (degC)

Return type:

numpy dictionary
